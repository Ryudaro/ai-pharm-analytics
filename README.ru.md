# AI-Pharm Analytics

AI-Pharm Analytics — это проект на ранней стадии, направленный на
прогнозирование спроса и аналитическое моделирование продаж
лекарственных препаратов в аптечных организациях.

Цель проекта — разработка локальной системы прогнозирования,
ориентированной на реальные условия работы аптек, где критически
важны конфиденциальность данных, интерпретируемость моделей и
воспроизводимость расчётов.

## Область применения

- Прогнозирование спроса на лекарственные препараты и аптечные товары
- Анализ временных рядов и статистическое моделирование
- Использование методов машинного обучения для прогнозирования спроса
- Локальное развёртывание без обязательного использования облачной инфраструктуры

## Текущий статус

Проект находится на ранней стадии разработки.

На текущем этапе репозиторий сфокусирован на:
- формализации задачи прогнозирования,
- методологической проработке,
- проектировании системы и архитектуры.

Детали реализации, наборы данных и производственный код
намеренно не публикуются на данном этапе.

## Планируемые направления развития

- Классические модели временных рядов (SARIMA и родственные подходы)
- Модели машинного обучения (например, градиентный бустинг)
- Оценка качества прогнозов на реальных аптечных данных
- Воспроизводимые и прозрачные пайплайны прогнозирования

## Примечания

Данный репозиторий представляет собой исследовательскую и инженерную
работу в процессе развития. Структура и содержимое проекта будут
эволюционировать по мере дальнейшей разработки.
